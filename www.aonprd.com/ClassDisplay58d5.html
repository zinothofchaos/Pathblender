Warrior

Source PRPG Core Rulebook pg. 450
From Nethys: This is an NPC class and was not given a description.
Hit Die: d10.

Class Skills

The Warrior's class skills are Climb (Str), Craft (Int), Handle Animal (Cha), Intimidate (Cha), Profession (Wis), Ride (Dex), and Swim (Str).

Skill Points at each Level: 2 + Int modifier.
